# README

* Crear en el root un archivo .env con las siguientes variables:

- RAILS_ENV="development"

- DB_HOST="db"
- POSTGRES_DB="nombre_db"
- POSTGRES_USER="postgres"
- POSTGRES_PASSWORD="postgres"


* Ejecutar docker-compose up
